
# ✨ Smooth Image Gallery

A modern and responsive **Image Gallery** built using **HTML, CSS, and JavaScript**. This project features smooth scrolling animations, hover zoom effects, and intuitive navigation controls for a great user experience.

---

## 📌 Features

* 🖼️ Display images in a clean layout
* 🔄 Smooth scrolling animation while navigating
* ⏭️ Next & Previous navigation buttons
* 🔍 Hover effect (image zoom-in)
* 💡 Responsive and user-friendly design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox / Grid, Transitions, Animations)
* JavaScript (DOM Manipulation)

---

## 📂 Project Structure

```
image-gallery/
│── index.html
│── style.css
│── script.js
│── images/
│   ├── img1.jpg
│   ├── img2.jpg
│   └── ...
```

---

## ⚙️ How It Works

* Images are displayed in a scrollable container.
* JavaScript controls the smooth scrolling behavior.
* Navigation buttons allow users to move forward and backward.
* CSS hover effects increase image size for better preview.
